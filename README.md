# dotfiles

Collection of my `vim`, `zshell`, and `brew` configurations.

## Symlink

```bash
ln -s "$(pwd)/.vimrc" ~/.vimrc
```

## Brew Installs

```bash
brew install gh
brew install typos-cli
brew install mysql-client

# Virtual Environments
brew install pipx
brew install nvm
brew install rbenv
brew install direnv

# Kubernetes
brew install k9s
brew install kubectx
brew install kube_ps1
```
